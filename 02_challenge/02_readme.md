# 02 challenge

## описание ошибки
в точке входа при вызове функции `fizzbuzz()` не передавался обязательный позиционный аргумент `max_num`, определенный в сигнатуре функции. это вызывало ошибку `TypeError`.

## решение
передали числовой аргумент `100` при вызове функции: `fizzbuzz(100)`.
теперь программа выводит корректную последовательность чисел.

## что делает функция
функция `fizzbuzz(max_num)` выводит числа от 1 до `max_num - 1` по правилам fizzbuzz (кратные 3 заменяются на `fizz`, кратные 5 — на `buzz`, кратные обеим — на `fizzbuzz`).
