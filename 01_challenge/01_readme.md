# 01 challenge

## описание ошибки
в строке `elif i%num1=0:` использовался одиночный знак равенства `=`, который является оператором присваивания, а не сравнения. это вызывало синтаксическую ошибку `SyntaxError`.

## решение
заменили оператор присваивания `=` на оператор сравнения `==`, чтобы правильно проверять остаток от деления.
теперь строка выглядит так: `elif i%num1==0:`.
