# 04 challenge

## описание ошибки
в блоке `if i%num1==0 and i%num2==0:` инструкция `print(i,three_mul+five_mul)` имела неверное количество отступов (8 пробелов вместо 12). это вызывало ошибку `IndentationError: expected an indented block`.

## решение
исправили отступы перед `print` на 12 пробелов (или 3 таба/уровня отступов), чтобы он правильно находился внутри блока `if`.
теперь код запускается и работает верно.

## что делает функция
функция `fizzbuzz(max_num)` производит вывод чисел от 1 до `max_num - 1` с заменой кратных 3 на `fizz`, кратных 5 на `buzz`, а кратных 3 и 5 на `fizzbuzz`.
