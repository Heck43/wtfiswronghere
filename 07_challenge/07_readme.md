# 07 challenge

## описание ошибки
функция `fizzbuzz(99)` вызывалась со значением `99`. в цикле `for i in range(1, max_num)` это приводило к тому, что проверялись числа только от 1 до 98 включительно. число 99 исключалось, что нарушало условие задачи "включая оба".

## решение
изменили аргумент вызова функции на `100`: `fizzbuzz(100)`.
теперь цикл проходит по всем числам от 1 до 99.
