# 10 challenge

## описание ошибки
в строке `import fizzbuzz` импортировался одноименный модуль. затем выполнялась попытка вызова модуля напрямую `fizzbuzz(100)`, что приводило к ошибке `TypeError: 'module' object is not callable`, так как модуль сам по себе не является вызываемым объектом.

## решение
заменили вызов модуля на вызов функции `fizzbuzz` внутри этого модуля: `fizzbuzz.fizzbuzz(100)`.
теперь импортированная функция вызывается корректно.
