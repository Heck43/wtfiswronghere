# 06 challenge

## описание ошибки
в строке `num2 = conf.num` пытались обратиться к несуществующему полю `num` в модуле `conf.py` (там определены только `num1` и `num2`). это приводило к ошибке `AttributeError: module 'conf' has no attribute 'num'`.

## решение
заменили обращение к `conf.num` на существующую переменную `conf.num2`.
теперь импортируемые параметры работают без ошибок.

## что делает функция
функция `fizzbuzz(max_num)` берет значения делителей `num1` и `num2` из импортированного конфигурационного модуля `conf.py` и на их основе выполняет вывод fizzbuzz.
