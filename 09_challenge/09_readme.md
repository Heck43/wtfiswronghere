# 09 challenge

## описание ошибки
в цикле `for i in range(6, max_num)` начальное значение диапазона было установлено в `6`, из-за чего программа пропускала первые числа (1, 2, 3, 4, 5). это приводило к пропуску `3 fizz` и `5 buzz` на выводе.

## решение
изменили начальный индекс цикла с `6` на `1`: `for i in range(1, max_num)`.
теперь цикл начинается с 1 и программа корректно выводит все значения.
