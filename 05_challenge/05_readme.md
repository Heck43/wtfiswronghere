# 05 challenge

## описание ошибки
1. использовалось неверное имя файла `mifile.txt`, тогда как файл на самом деле называется `myfile.txt` (`FileNotFoundError`).
2. использовался синтаксис вывода python 2: `print 'i have created'`, что приводило к `SyntaxError` в python 3.

## решение
1. исправили имя файла в `open()` на `myfile.txt`.
2. обернули выражение в принт скобками: `print('i have created')`.
теперь программа считывает параметры из файла и запускается корректно.
