# 03 challenge

## описание ошибки
при вызове функции `fizzbuzz('16')` в качестве аргумента передавалась строка, а не целое число. при выполнении `range(1, max_num)` это приводило к ошибке `TypeError: str object cannot be interpreted as an integer`.

## решение
заменили строковый аргумент `'16'` на целочисленный `100` для корректного выполнения цикла `range(1, 100)`.
теперь программа работает правильно.

## что делает функция
функция `fizzbuzz(max_num)` принимает целое число `max_num` и выполняет вывод последовательности fizzbuzz от 1 до `max_num - 1` включительно.
